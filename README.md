# Jobkita

Another job Listing Portal.

## Feature List

    1. Homepage
        - Job Search Form
            - Keyword
            - Location
        - Job Posting Button

    2. Job Listing
        - Show jobs based on current Location
        - Show company name, logo, position, location, and tags on job listing item
        - Job filter section
        - Sort job (oldest or newest)

    3. Job Details
        - Company Details
        - Job description
        - Date posted

    4. Job Posting
        - Show job listing Form
            - Company name
            - Address
            - Job position
            - Job description

## Stacks

- React
- Typescript
- TailwindCSS
- HeadlessUI
- MomentJS

## Run Locally

Clone the project

```bash
  git clone https://github.com/sudiddo/jobkita.git
```

Go to the project directory

```bash
  cd jobkita
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

## Running Tests

To run tests, run the following command

```bash
  yarn test
```
